# NativeAnalyticsBehavior

Behavioral analytics companion for the NativeAnalytics ProcessWire module.
Phase 1: click + scroll-depth heatmaps.

## Install
1. Copy this folder to `/site/modules/NativeAnalyticsBehavior/`.
2. Modules > Refresh.
3. Install **NativeAnalyticsBehavior** and **NativeAnalyticsBehavior Dashboard**.
4. Configure under Modules > Configure > NativeAnalyticsBehavior.
